> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a Minimum Viable Product (MVP) for the Q-TAIL-MVP project, focusing on a quantum-guided long-tail task scheduler on Meta-World MT10 to improve tail task success and CVaR@20.

**Architecture:** A 5-agent orchestrator system (Quantum Source, Semantic Mapper, Quantum Scheduler, Training, Evaluation) that reads pre-computed quantum distributions and dynamically schedules MT10 tasks without modifying the underlying RL policy or environment source code. 

**Tech Stack:** Python, Meta-World (local `metaworld/` directory), Numpy, Matplotlib, Pandas, PyYAML.
